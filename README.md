# Equations of a Living Planet

**Subtitle:** The Energy Rules Behind Earth’s Chemical Cycles

This repo is a Quarto book written in Markdown. It’s intended to be:

- **Readable on GitHub** (chapters are `*.md`)
- **Readable as a website** via **GitHub Pages**
- **Downloadable** as **PDF** (and **EPUB**)

## Read online
After the GitHub Pages workflow runs, the site will be available under your repository’s Pages URL (see GitHub repo settings: **Settings → Pages**).

## Download
Each workflow run uploads `downloads` artifacts containing the **PDF** and **EPUB**.

## Local build
Install Quarto, then render:

```bash
quarto render --to html,pdf,epub
```

The outputs will be in `_book/`.
